## Important Finding
The previous zero-byte output issue was not only a UI display problem. Root cause:
- SIT and some UI runs used `route_mode=targeted` with only `ingestion`.
- The pipeline completed successfully because ingestion completed.
- `memo_architect` did not run, so `final_state.draft_memo` was empty.
- `main.py` wrote that empty value to a `.txt` artifact.

Current behavior after fix:
- If `draft_memo` is empty, `main.py` writes a clearly labelled partial report.
- This prevents silent blank output.
- It also tells the user to rerun with `memo_architect` or full pipeline if a final memo is needed.
